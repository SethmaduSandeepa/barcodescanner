# Barcode Scanner System

This is a full-stack barcode management system for IT asset tracking.

## Features
- Create, scan, and print barcodes
- Professional asset details display
- Prevent duplicate entries
- Print layout with landscape orientation
- Update barcode details

## Tech Stack
- React (frontend)
- Node.js/Express (backend)
- MongoDB (database)

## Getting Started
1. Clone the repo
2. Install dependencies in `client` and `server`
3. Start backend: `npm start` in `server`
4. Start frontend: `npm start` in `client`
